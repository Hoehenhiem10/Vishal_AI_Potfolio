Artifact 1 Reflection

Customization for Target Audience

This artifact was carefully designed for three distinct audiences:

  • Non-technical users: Created intuitive interfaces with clear visual elements using Canva, ensuring accessibility without requiring AI expertise
  • Fintech recruiters: Demonstrated practical AI implementation through working prototypes and comprehensive documentation
  • AI/ML professionals: Included detailed model comparisons with performance metrics and prompt engineering methodologies

The three-tiered approach ensured each audience could engage with the content at their comfort level while extracting relevant value.

Lessons Learned

Technical Insights:

  • Prompt specificity dramatically improves output quality - adding context like risk tolerance scales transformed generic advice into personalized recommendations
  • Each LLM has distinct strengths: GPT-4o excels at structured breakdowns, Claude provides nuanced risk assessments, Gemini offers creative strategies
  • UI/UX integration requires paradigm shifts - users expect instant calculations but LLMs work better conversationally

Process Discoveries:

  • Conversational AI reduces financial anxiety barriers that prevent engagement with traditional tools
  • Visual outputs significantly increase user trust in AI-generated financial advice
  • Robust error handling is crucial for maintaining user confidence

Revisions Applied

Through iterative development based on user testing:

  • Enhanced Personalization: Added demographic inputs and financial goal categorization for targeted advice generation
  • Comparative Analysis Matrix: Created detailed comparison tables with specific metrics for each model tested
  • Visual Improvements: Replaced low-resolution screenshots with annotated high-quality images and user journey flow diagrams
  • Security Section: Added comprehensive data handling explanations including local processing options
  • Contextualized Timeline: Enhanced AI evolution timeline with practical implications annotations

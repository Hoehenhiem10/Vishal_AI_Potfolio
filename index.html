🧠 Artifact 1: Applied AI Design – From Prompt to Product

🔹 Introduction

This artifact showcases a hands-on project focused on designing a custom AI assistant to simplify financial planning for everyday users. The project demonstrates my ability to build a usable tool from scratch using prompt engineering, UI design, and app development.

🎯 Objective

To create an accessible and intelligent chatbot that helps users plan savings based on their income, spending, and financial goals.

🧩 Description

I developed “BudgetBuddy,” a personalized GPT-based financial assistant. It was built to analyze user inputs (income, expenses, savings goals) and generate a savings plan. This tool integrates AI prompts, financial logic, and a friendly front-end interface.

🔧 Process

  • Researched budgeting methods and common financial planning needs.
  • Engineered prompts for savings plan logic in GPT.
  • Designed an intuitive UI prototype in Canva.
  • Deployed an interactive Streamlit app to simulate real-time use.
  • Created visual and clickable prototypes to demonstrate user flow.

🛠 Tools and Technologies Used

  • OpenAI ChatGPT
  • Streamlit (for app prototype)
  • Canva (for UI mockups and timeline)
  • Custom GPT configuration
  • GitHub Pages (portfolio host)

  • Communicate complex functionality through design and interaction

Combines LLM prompting, user interface design, and front-end logic into one coherent, working concept—demonstrating cross-functional competence.
